# Open-Source-Linux
Open Source Linux Project
Harshil Rajput 24BEC10144
The Open Source Audit capstone project explores the concept, importance, and practical use of open-source software. It combines theoretical understanding with hands-on experience using Linux shell scripting. The project includes an 11-page report, five shell scripts, and an analysis of one open-source software. The scripts perform useful system tasks such as displaying system information like kernel version, user, uptime, and distribution; checking if software is installed along with its version and license; analyzing disk usage, permissions, and ownership of directories; scanning log files to count errors and generate summaries; and creating a personalized open-source philosophy statement saved to a file. The project requires a Linux operating system such as Ubuntu or Fedora, along with a Bash shell and basic utilities like uname, uptime, grep, awk, du, and ls. Overall, the project helps in understanding open-source philosophy, licensing, real-world software ecosystems, and builds practical skills in Linux and shell scripting.
